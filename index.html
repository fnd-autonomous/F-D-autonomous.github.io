Project Overview

This project focuses on developing an autonomous driving algorithm capable of navigating complex environments. Our system integrates computer vision, sensor fusion, and path planning to create a robust self-driving solution.

The project utilizes Gazebo simulation for initial development and testing, providing a safe and cost-effective environment for algorithm validation before real-world deployment.

Methodology

System Architecture

Our autonomous driving system consists of three main components:

  • Perception Module: Processes sensor data for environment understanding
  • Planning Module: Generates optimal trajectories and decisions
  • Control Module: Executes planned actions through vehicle actuators

Simulation Environment

We are currently using Gazebo as our primary simulation platform, which provides:

  • High-fidelity physics simulation
  • Realistic sensor models (LIDAR, cameras, GPS, IMU)
  • Integration with ROS 2 for modular development
  • Customizable environments and scenarios
